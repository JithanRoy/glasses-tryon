# Glasses Try-on Monorepo

This project is a monorepo for a Glasses Try-on application.

## Structure

- `apps/frontend`: Next.js application
- `apps/backend`: NestJS application
- `packages/shared`: Shared types and interfaces

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development servers:
   ```bash
   npm run dev
   ```
